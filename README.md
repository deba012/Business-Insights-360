## 🚀 Project Overview

Business Insights 360 integrates data from SQL databases and Excel files to provide actionable insights across multiple business functions. The solution consists of interactive dashboards for Finance, Sales, Marketing, Supply Chain, and Executive Management, enabling stakeholders to monitor KPIs, analyze trends, and improve business performance.

## 🛠️ Tech Stack

- Power BI
- SQL
- Excel
- Power Query

## 📈 Key Features

- Built **5 interactive dashboards** for different business departments
- Integrated data from **SQL and Excel** sources
- Analyzed **1M+ records** using an optimized data model
- Implemented drill-down and dynamic filtering capabilities
- Delivered actionable insights on sales, profitability, customers, products, and forecasting

## 📊 Dashboard Views

### Marketing View

### Supply Chain View

## 🔍 Key Business Insights

- APAC emerged as the highest revenue-generating region.
- Europe achieved the highest gross margin percentage.
- Notebook products contributed the largest share of revenue.
- Forecast accuracy improved compared to the previous year.
- Inventory risks were identified through forecast error analysis.
